											前处理功能简介


一、常用功能介绍
	- 独特的解决方案设计，实现多工程轻松管理
	- 与后处理软件交互，快速查看模型，快速查看求解的结果
	- 解决方案历史记录，快速实现模型文件的导入
	- 一键求解模型
    - 求解队列管理：可将无限数目的工程加入求解，可设置同时最大求解工程数
    - 对单个求解工程：查看求解相关信息，暂停、重启、停止，可转入后处理查看当前求解的结果
    - 实时模型结构树，快速导航到模型块
	- 编译查错功能，快速定位模型错误代码
	- 统一的设置管理，系统个性化定制，使用习惯保存
    - 内置使用帮助，可快速查询关键字条目用法
    - 内置自动更新程序

二、解决方案树主要功能
	- 工程文件夹、文件夹、标签分类、备注，多种方式进行工程分类管理
    - 管理所有模型相关文件，创建、导入、修改和删除文件，定位到文件或工程文件夹
	- 一键克隆工程，方便进行模型参数分析和调试
	- 参数敏感性分析：快速建立一批不同参数值的模型进行批量求解
	- 内置支持使用Subversion对解决方案进行版本控制
    - 支持使用内置的程序或自定义外部软件比较文件差异

三、编辑器主要功能
    - 关键字着色显示
	- 组件和体代码收缩
	- 智能关键字提示：条目关键字，书写特定条目时特定条目支持的属性，或列出某关键字支持的所有属性
	- 对于PART、ASSEMBLY、I、J等引用属性支持提示可引用条目
    - 快速转到实例和引用的定义
	- 可在编写模型时实时显示关键字帮助
	- 模型代码一键格式化，显示字体样式自定义
	- 支持同时查看比较多个模型文件
	- 关键字查找、替换功能
